 SupKrellM - Monitoring système Linux
Application développée en Python permettant de collecter des informations système Linux et de générer un rapport HTML détaillé.

 Objectif
Créer un outil de monitoring capable :
- de collecter des métriques système
- de générer un rapport HTML statique
- d’afficher les données en temps réel via une interface graphique
 Fonctionnalités
 Collecte des données système
- Date et heure du système
- Nom d’hôte
- Version du noyau Linux
- Uptime
- État de la mémoire (RAM)
- État des disques
- Liste des processus actifs
- État du réseau
- Températures système

 Génération de rapport HTML
- Rapport structuré et lisible
- Design responsive
- Gestion des erreurs intégrée

 Interface graphique (Tkinter)
- Affichage en temps réel
- Mise à jour automatique
- Interface fluide et non bloquante

 Robustesse
- Gestion complète des erreurs
- Aucun crash du programme
- Traitement intelligent des données

 Technologies utilisées
- Python 3 (standard library uniquement)
- Tkinter
- HTML5 / CSS3

 Utilisation
 Générer un rapport :
python main.py
